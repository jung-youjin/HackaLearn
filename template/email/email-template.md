# email-template

이메일에 사용되는 템플릿 뼈대는 모두 동일하며, 절차마다 수정되어야하는 부분은 아래와 같다
- `<img id ="banner" src="">`의 이미지 배너 링크
- `<tbody id="contents-body">`의 메일 내용 부분
    ``` html
    <div id="dear">
    ... 
    </div>

    <div id="list">
    ...
    </div>

    <div id="detail">
    ...
    </div>

    <div id="ending">
    ...
    </div>

    <div id="from">
    ...
    </div>
    
    <div id="button">
    ...
    </div>
    ```
